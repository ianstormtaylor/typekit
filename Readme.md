# typekit
  
  Load a Typekit kit asynchronously, automatically handling FOUT.

## Installation

    $ component install ianstormtaylor/typekit

## Example

```js
var typekit = require('typekit');
typekit('yzu2jpo');
```

## API

### typekit(id, options)

  Load a kit with `id` and optional `options`:

    {
      timeout: 3000
    }
